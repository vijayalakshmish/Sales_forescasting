# Sales_forescasting
Sales Forecasting
A data science project focused on predicting future product sales using machine learning in Python based on advertising spend across multiple media channels.

## Project Structure
Plaintext

├── README.md                           # Project documentation
└── Sales_Prediction_Using_Python.ipynb # Google Colab / Jupyter Notebook workflow

## Dataset Features
The model analyzes advertising budgets across three primary channels to forecast sales:

TV: Advertising budget spent on TV.

Radio: Advertising budget spent on radio.

Newspaper: Advertising budget spent on newspapers.

Sales: The target variable (predicted sales units).

## Quick Start
Clone the repository:

Bash
git clone https://github.com/vijayalakshmish/Sales_forescasting.git
cd Sales_forescasting
Install required libraries:

Bash
pip install numpy pandas matplotlib seaborn scikit-learn
Run the project:
Open Sales_Prediction_Using_Python.ipynb in Jupyter Notebook or upload it directly to Google Colab to execute the cells
